# 🎯 Início Rápido - Comece Agora Mesmo

## 1. Instale (1 minuto)

No terminal do Claude Code:
```bash
cd c:\Users\emanu\Documents\Projetos\multiagentes
pip install -e .
```

## 2. Teste Imediato (30 segundos)

No chat do Claude Code, digite:
```
/plano Criar uma função Python que valida e-mail
```

## 3. See the Magic ⚡

O sistema vai:
- ✅ Gerar um plano estratégico
- ✅ Validar o plano
- ✅ Verificar dependências
- ✅ Retornar plano pronto
